## 🚀 Getting Started
1. Clone the repo:
```bash
git clone https://github.com/st3adyp1ck/portfolio.git
```
2. Serve locally:
```bash
python -m http.server
```
3. Visit `http://localhost:8000` [[3]][[8]]
