St3adyP1ck's Simple Calculator

Notes & Instructions

Welcome to the Simple Calculator! This tool allows you to perform simple addition using JavaScript. It’s designed to be straightforward and easy to use. Just input two numbers, and the calculator will return their sum. If one of the fields is left blank, you'll be prompted to enter a number.

How To Use:

  1. Enter Numbers: Provide a number in both input fields.
  2. Calculate Result: Click the ‘Calculate’ button to display the result.
  3. Mandatory Input: Both fields must contain numbers. An error message will appear if either field is left blank.

JavaScript Methods Used:

  • DOM Manipulation:
    • document.getElementById(): retrieves elements by id
    • textContent: updates text content
    • classList.add()/remove(): manages CSS classes dynamically.
  • Event Handling: The onclick method triggers a specified function when the button is clicked, handling user interaction.
  • Type Conversion: Number() converts strings to numbers for calculations, and parseInt()/parseFloat() handle integer and decimal conversion.
  • Conditional Logic: if statements control the logic flow, and isNaN() validates if inputs are not numbers.
  • String Handling: Template Literals (${}) allow embedding expressions into strings, making concatenation and formatting easier.
