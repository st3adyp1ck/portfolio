"Guess a Number" Mini-Game

Guess My Number is a fun, interactive JavaScript-based game where the player tries to guess a randomly generated number between 1 and 20.

JavaScript Methods Used:

  • Strict Mode & DOM Manipulation:
    • 'use strict'; - Enables strict mode, catching common coding mistakes.
    • document.querySelector() - Selects elements to manipulate.
    • .textContent, .value, and .style - Used to get or set text, input values, and styles.
  • Event Listeners & Functions:
    • addEventListener('click', function() {}) - Adds click event listeners for interactivity.
    • Arrow functions => and helper functions like displayMessage() encapsulate repeated logic.
  • Math & Game Logic:
    • Math.random() and Math.trunc() - Used to generate random numbers.
    • if/else if statements and ternary operators handle game logic (e.g., comparing guesses, winning, or losing).
  • State Management & Reusability:
    • Global variables (score, secretNumber, highScore) maintain game state.
    • Reusability through functions (displayMessage(), updateScore()) to avoid repetitive code.

"Pig Game" Mini-Game

Pig Game is a two-player dice game built using JavaScript. Players take turns rolling a dice to accumulate points.

JavaScript Methods Used:

  • Strict Mode & DOM Manipulation:
    • 'use strict'; - Enables strict mode to catch errors.
    • document.querySelector() and document.getElementById() - Used to select and manipulate HTML elements like player scores, current score, and dice image.
    • .textContent and .style - Modify the text content and styles of elements (e.g., changing active player visuals).
  • Event Listeners:
    • addEventListener() - Listens for user actions (e.g., roll dice, hold score, reset game).
  • Game Logic with Conditional Statements:
    • if/else if - Handle game flow (e.g., checking if a player rolls a 1, switching turns, checking for a winner).
  • Random Dice Rolls:
    • Math.random() and Math.trunc() - Used to simulate random dice rolls between 1 and 6.
  • State Management & Game Reset:
    • Global variables (scores, currentScore, activePlayer) manage the game state across player turns and game resets.
    • Functions reset the game state and handle switching between players.

Simple Calculator

This tool allows you to perform simple addition using JavaScript. It’s designed to be straightforward and easy to use. Just input two numbers, and the calculator will return their sum. If one of the fields is left blank, you'll be prompted to enter a number.

JavaScript Methods Used:

  • DOM Manipulation:
    • document.getElementById(): retrieves elements by id
    • textContent: updates text content
    • classList.add()/remove(): manages CSS classes dynamically
  • Event Handling: The onclick method triggers a specified function when the button is clicked, handling user interaction.
  • Type Conversion: Number() converts strings to numbers for calculations, and parseInt()/parseFloat() handle integer and decimal conversion.
  • Conditional Logic: if statements control the logic flow, and isNaN() validates if inputs are not numbers.
  • String Handling: Template Literals (${}) allow embedding expressions into strings, making concatenation and formatting easier.
